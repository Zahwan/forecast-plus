Forecast Plus
Provides local and long range weather forecast
Dimensions: Popup width is px.

Acceptable popup width range is 300px to 800px.

Popup height is px.

Acceptable popup height range is 300px to 600px.
Badge icon: Check weather condition every minutes.

Minimum period is 1 minute.
Badge number: Use decimal point in the badge number:

By default, the current temperature is round to the nearest whole number. Check this option if you prefer the extension to display decimal point as well. Wait for the next update for this change to take place.
